Документация vk_api
===================

vk_api – Python модуль для написания скриптов для социальной сети Вконтакте
(vk.com) (API wrapper)

`Установка через PIP
<https://pythonworld.ru/osnovy/pip.html>`_:

.. code-block:: bash

   $ pip install vk_api
